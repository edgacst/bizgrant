# GrantHunter — 정부지원금/RFP 알리미 기획서

## 1. 제품 개요

| 항목 | 내용 |
|------|------|
| 제품명 | **GrantHunter** (그랜트헌터) |
| 슬로건 | "당신의 돈, 놓치지 마세요" — 정부지원금 AI 매칭 서비스 |
| 타겟 | 중소기업, 스타트업, 소상공인, 프리랜서, 연구기관 |
| 가치제안 | 흩어진 정부지원 공고를 업종·규모별로 필터링, 마감 직전까지 알림 |
| 비즈니스모델 | SaaS 월 구독 (Free / Pro / Enterprise) |

---

## 2. 핵심 기능

### 2.1 공고 수집
| 소스 | 방식 |
|------|------|
| 나라장터 g2b.go.kr | RFP/RFQ 공고 스크래핑 |
| 기업마당 bizinfo.go.kr | 중소기업 지원사업 |
| 중소벤처기업부 mss.go.kr | 창업·R&D·수출 지원 |
| K-Startup k-startup.go.kr | 스타트업 지원 |
| 지자체 홈페이지 | 지역별 지원사업 |
| 과기정통부/산업부 등 | 부처별 R&D 공고 |

### 2.2 AI 매칭
- 업종 태깅 (제조/IT/서비스/유통 등 20개 카테고리)
- 지원자격 자동 파싱 (매출액, 업력, 지역, 인원)
- 사용자 프로필과 교차 매칭 → 적합도 점수
- "붙을 확률" 예측 (Feasibility Score)

### 2.3 알림 체계
| 트리거 | 채널 |
|--------|------|
| 신규 공고 매칭됨 | Telegram, Slack, Webhook |
| 마감 D-7/D-3/D-1 | 재알림 |
| 공고 내용 변경 | 업데이트 알림 |
| 유사 공고 추천 | 주간 다이제스트 |

### 2.4 부가 기능
- **체크리스트**: 지원에 필요한 서류 자동 리스트업
- **캘린더 동기화**: 마감일 Google Calendar/iCal 연동
- **성공사례 DB**: 비슷한 기업의 수주 성공케이스
- **컨설팅 매칭**: 공고별 전문 컨설턴트 연결 (제휴)

## 8. 차별화 포인트

1. **속도**: 정부사이트보다 먼저 알림 도착 (주기적 스크래핑)
2. **AI 매칭**: 단순 키워드가 아닌 자격요건을 이해하고 매칭
3. **올인원**: 10개 이상 기관을 한 곳에서 모니터링
4. **액션 가이드**: "이 공고에 붙으려면 뭘 준비해야 하나"까지 제공
